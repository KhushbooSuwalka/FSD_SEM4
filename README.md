## 🚀 How to Run the Project

### 🔹 Frontend

1. Navigate to the frontend folder:

   ```
   cd folder-name
   ```
2. Install dependencies:

   ```
   npm install
   ```
3. Run the project:

   ```
   npm start
   ```

---

### 🔹 Backend

1. Navigate to backend folder:

   ```
   cd server
   ```
2. Install dependencies:

   ```
   npm install
   ```
3. Start server:

   ```
   node index.js
   ```

## 📚 Experiments Included

1. **Develop a static HTML page** with text formatting, tables, headers, links, images, and lists.

2. **Create a webpage using CSS**

   * Apply internal and external CSS
   * Design layout using Tailwind CSS
   * Implement forms and dropdown components

3. **Implement JavaScript interactivity**

   * Create alerts, buttons, and popovers
   * Use AJAX to fetch and display data from an API

4. **DOM Manipulation using JavaScript**

   * Update content and styles dynamically
   * Handle user interactions

5. **Use jQuery and ES6 features**

   * Handle events
   * Manipulate HTML elements
   * Work with JSON data

6. **Build a React Functional Component**

   * Use JSX
   * Manage state
   * Handle events

7. **Implement React Router**

   * Create a Single Page Application (SPA)
   * Add multiple routes and navigation

8. **Node.js File System Operations**

   * Read, write, and manipulate files
   * Use built-in Node.js modules

9. **Create a RESTful API using Express.js**

   * Implement routes and middleware
   * Handle form data

10. **MongoDB Integration using Mongoose**

* Connect Node.js with MongoDB
* Perform CRUD operations

11. **Full Stack Application Development**

* Integrate React frontend + Node.js backend + MongoDB
* Implement authentication
* Add data storage and interactive features

This repository contains the practical experiments and projects developed as part of the **Full Stack Development (FSD) course in the 4th semester**. This project serves as an academic record as well as a foundational step towards developing real-world web applications.
